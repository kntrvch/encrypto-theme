Mobile animation

To disable mobile wireframe animation follow these steps:

1. Remove the <div> element with id attribute #mobile 

<div class="d-none d-md-block" id="mobile">
  <object id="mobile-svg" type="image/svg+xml" data="img/mobile-svg.svg"></object>
</div>
Copy

2. Add .visible class to the #mobile-image element 

<img src="img/mobile-image.png" class="d-none d-md-block visible" id="mobile-image" alt="">

Parallax effect

ENCRYPTO template comes with Parallax.js plugin which creates an element of depth and distance giving your website three-dimension feel.

You can set the parallax effect to all unskewed sections. If you need further info on plugin configuration, please visit Parallax.js website 

<section class="parallax-window" id="sale" data-parallax="scroll" data-image-src="img/bg-sale.jpg">

Components

Ticker

Ticker is a web component that allows you to have items scrolling indefinitely across the screen. In most cases it is used to display currency rates, trends and other info.

You need to have the following HTML structure:

<!--Ticker-->
<div class="ticker ticker--top" id="ticker">

  <!--.ticker__item-->
  <div class="ticker__item">
  ...
  </div>
  <!--/.ticker__item-->
  <!--.ticker__item-->
  <div class="ticker__item">
  ...
  </div>
  <!--/.ticker__item-->
  <!--.ticker__item-->
  <div class="ticker__item">
  ...
  </div>
  <!--/.ticker__item-->
    
</div>
Copy

There are two possible locations for the ticker - top and bottom - use <div class="ticker ticker--top" id="ticker"> and <div class="ticker ticker--bottom" id="ticker"> respectively.

Countdown

You can set end date via data-end attribute

<div class="countdown p-3 text-center wow fadeInUp" data-end="10/31/2018 12:00:00 AM">
  ...
</div>
Copy

It is important to provide date in format dd/MM/yyyy hh:mm:ss a

Charts

ENCRYPTO uses morris.js library to render beautiful and 100% responsive charts.

By default there are two charts in the template:

  • - distribution donut chart
  • - trending line chart

You can customize them via JS code in js/encrypto.js file


Donut chart


Morris.Donut({
  element: 'distribution-chart',
  data: [
    { label: "Download Sales", value: 12 },
    { label: "In-Store Sales", value: 30 },
    { label: "Mail-Order Sales", value: 20 }
  ],
  resize: true,
  colors: ['#ff6f00', '#6610f2', '#e83e8c', '#f0ad4e', '#5cb85c', '#5bc0de'],
  formatter: function (y, data) { return y + '%' }
});
Copy


Line chart


Morris.Line({
  element: 'trending-chart',
  data: [
    { y: '2012', a: 0, b: 0 },
    { y: '2013', a: 10, b: 0 },
    { y: '2014', a: 25, b: 10 },
    { y: '2015', a: 50, b: 15 },
    { y: '2016', a: 80, b: 40 },
    { y: '2017', a: 95, b: 65 },
    { y: '2018', a: 100, b: 75 }
  ],
  xkey: 'y',
  ykeys: ['a', 'b'],
  labels: ['Encrypto', 'Average'],
  hideHover: "auto",
  smooth: false,
  lineColors: ['#ff6f00', '#1a237e'],
  resize: true
});
Copy

For more information please refer to morris.js documentation
