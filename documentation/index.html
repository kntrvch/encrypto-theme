Menu

To edit the main menu, you have to look for the <ul class="navbar-nav ml-auto mr-3">...</ul> element

<ul class="navbar-nav ml-auto mr-3">...</ul>
Copy
  • - Each menu item links to a section (see section Sections below) in the main content.
  • - Each menu item should follow this format:
<li class="nav-item">
  <a class="nav-link" href="#contact">Contact</a>
</li>
Copy
Static navbar

To make the navbar static add .navbar-static CSS class to <body></body> element, like this:

<body id="page-top" class="js home navbar-static">
Copy

Header

Header animation is the first thing people see when they visit your website. It should draw their attention to your business and give visitors the best first impression possible.

There are three background animations available:

  • - gradient
  • - flat surface shader (FSS)
  • - particles effect

Gradient background


<!--BACKGROUND EFFECT-->
  <div class="gradient-bg" id="gradient"></div>
<!--/BACKGROUND EFFECT-->
Copy

FSS background


<!--BACKGROUND EFFECT-->
<!--.fss-container-->
<div id="background-container" class="background-container">
  <div id="background-output" class="background-output"></div>
</div>
<!--/.fss-container-->
<!--/BACKGROUND EFFECT-->
Copy

Particles background


<!--BACKGROUND EFFECT-->
  <div id="particles-js"></div>
<!--/BACKGROUND EFFECT-->
Copy

Mobile animation

To disable mobile wireframe animation follow these steps:

1. Remove the <div> element with id attribute #mobile 

<div class="d-none d-md-block" id="mobile">
  <object id="mobile-svg" type="image/svg+xml" data="img/mobile-svg.svg"></object>
</div>
Copy

2. Add .visible class to the #mobile-image element 

<img src="img/mobile-image.png" class="d-none d-md-block visible" id="mobile-image" alt="">
Copy

Sections

The theme includes three types of sections:

  • - normal
  • - left-skewed
  • - right-skewed


Normal section


<section>
  <!--section content-->  
   ...
  <!--/section content-->    
</section>
Copy

Left-skewed section


<section class="skew skew--left">
  <div class="section__skew--top">
    <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100" preserveAspectRatio="none">
      <polygon points="0,100 0,0 100,100" />
    </svg>
  </div>
  <!--section content-->  
  ...
  <!--/section content-->  
  <div class="section__skew--bottom">
    <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100" preserveAspectRatio="none">
      <polygon points="0,0 100,0 100,100" />
    </svg>
  </div>
</section>
Copy

Right-skewed section


<section class="skew skew--right">
  <div class="section__skew--top">
    <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100" preserveAspectRatio="none">
      <polygon points="0,100 100,0 100,100" />
    </svg>
  </div>
  <!--section content-->  
  ...
  <!--/section content-->  
  <div class="section__skew--bottom">
    <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100" preserveAspectRatio="none">
      <polygon points="0,0 100,0 0,100" />
    </svg>
  </div>
</section>
Copy

When applying custom background color to skewed section, remember to set proper fill attribute for both SVG elements: 


section.skew#contact .section__skew--top svg, 
section.skew#contact .section__skew--bottom svg {
  fill: black;
}
Copy

Parallax effect

ENCRYPTO template comes with Parallax.js plugin which creates an element of depth and distance giving your website three-dimension feel.

You can set the parallax effect to all unskewed sections. If you need further info on plugin configuration, please visit Parallax.js website 

<section class="parallax-window" id="sale" data-parallax="scroll" data-image-src="img/bg-sale.jpg">
Copy
